Your view must meet the following requirement:
  • "Items to display" in your basic view settings must be set higher than the "Items to show" value in your spy options.
Example setup:
Choose Views Spy in the Style dialog within your view, which will prompt you to configure:
  • Items to show: How many items to show all the time. Must be lower than the items being shown by the view.
  • Transition time: The time in seconds to wait before switching the next item in. Note that this will be added to Fading speed and Sliding speed for the total value that will be used.
  • Fading speed: The time in seconds for the fading in and out of the elements.
  • Sliding speed: The time in seconds for the sliding down of the elements.
  • Pause on mouse over: Whether or not the Spy should pause when the user's mouse hovers the section.
  • Use the module's default styling: If you disable this, the file in the module's directory views-spy.css will not be loaded
